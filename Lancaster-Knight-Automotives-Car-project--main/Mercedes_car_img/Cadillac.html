Cadillac Cars Overview

Cadillac Escalade

Engine: 6.2L V8 Petrol / 3.0L Turbo Diesel / 6.2L V8 with Supercharger (V-Series)
Transmission: 10-speed Automatic
Fuel Type: Petrol / Diesel
Body Type: Full-Size Luxury SUV (High-End Comfort, Advanced Tech, Premium Interiors)
Mileage: Petrol: 6–7 km/l, Diesel: 9–10 km/l (approximate, depending on variant)

Cadillac Escalade ESV

Engine: 6.2L V8 Petrol / 3.0L Turbo Diesel / 6.2L V8 Supercharged (V-Series)
Transmission: 10-speed Automatic
Fuel Type: Petrol / Diesel
Body Type: Full-Size Extended Luxury SUV (Extra Cargo Space, Premium Features)
Mileage: Petrol: 5.5–6.5 km/l, Diesel: 8.5–9.5 km/l (approximate, depending on trim)

Cadillac Escalade V-Series

Engine: 6.2L Supercharged V8 Petrol (682 hp)
Transmission: 10-speed Automatic
Fuel Type: Petrol
Body Type: High-Performance Luxury SUV (Blends Power, Prestige, and Premium Comfort)
Mileage: Petrol: 5–6 km/l

Cadillac CT6

Engine: 2.0L Turbo Petrol / 3.6L V6 Petrol / 4.2L Twin-Turbo V8 (CT6-V)
Transmission: 10-speed Automatic (8-speed on earlier models)
Fuel Type: Petrol
Body Type: Full-Size Luxury Sedan (Executive Class, Tech-Rich, Comfort-Oriented)
Mileage: 8–12 km/l

Cadillac CT5

Engine: 2.0L Turbo Petrol / 3.0L Twin-Turbo V6 (available in CT5-V and Blackwing)
Transmission: 10-speed Automatic, 6-speed Manual (available in CT5-V Blackwing)
Fuel Type: Petrol
Body Type: Midsize Luxury Sedan (Sporty Dynamics, Premium Interiors, Advanced Tech)
Mileage: 8–12 km/l
